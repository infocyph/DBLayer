Installation
============

Requirements
------------

- PHP ``^8.4``
- ``ext-pdo``
- Optional per driver:
  - ``ext-pdo_mysql``
  - ``ext-pdo_pgsql``
  - ``ext-pdo_sqlite``

Install
-------

.. code-block:: bash

   composer require infocyph/dblayer

Development Dependencies
------------------------

.. code-block:: bash

   composer install
